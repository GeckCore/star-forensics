# Changelog

All notable changes to star-forensics are documented here.

## [0.1.0] — 2025-01-01

### Added
- Initial release
- 8 detection patterns: Ghost Accounts, Account Creation Clustering, Star Velocity Spikes,
  Freshly Minted Accounts, Socially Isolated Accounts, Bot-like Usernames,
  Repoless Stargazers, Star/Fork Disproportion
- Trust Score (0–100) with confidence levels
- Rich terminal output
- JSON and HTML report export
- Python API
- GitHub Actions CI/CD pipeline
